# Birthday Gift Website

Website birthday gift satu halaman untuk Pricill, dibuat dengan React, Vite, Tailwind CSS, Framer Motion, Lucide React, dan Canvas Confetti.

## Cara Install

```bash
npm install
```

## Cara Menjalankan

```bash
npm run dev
```

## Cara Build

```bash
npm run build
```

## Cara Preview Build

```bash
npm run preview
```
